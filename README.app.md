# Template Project

A React + Vite starter for local automation tools, internal dashboards, and workflow UIs.

## Getting started

```bash
npm install
npm run dev
```

## Available scripts

```bash
npm run dev
npm run build
npm run preview
npm run typecheck
npm run lint
npm run format
npm run test
```

## Stack

- React
- Vite
- TypeScript
- Zustand
- React Router
- PostCSS
- Autoprefixer
- Vitest
- Testing Library
- Prettier
- ESLint 9
